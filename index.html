Control Panel

The control panel is the initial setup for the bluetooth low energy slaves addesses that will be connected to the hub. Before a new entry or modifying any bluetooth address the user must stop the Hub-Daemon. Then the user must insert o update the addresses using the write button. The read button also reads the data in the sql data base. When you finish writing or updating the addresses, you must re-start the Hub Daemon again.
